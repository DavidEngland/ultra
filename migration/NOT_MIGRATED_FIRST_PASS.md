# Not Migrated in First Pass

This file documents intentionally deferred content from the parent ABL workspace.

## Deferred categories

- Broad manuscript archives outside direct ultraspherical execution path
- Personal/admin communication files
- Legacy backups and temporary scratch files
- Large historical output trees that can be regenerated
- Optional language stacks (Fortran and Python utilities) not required for Julia-first MVP

## Suggested second-wave migration

1. Fortran integration modules under examples/
2. Python utility modules under code/
3. Additional notebook teaching materials under notebooks/ and hw/
4. Consolidated bibliography and references package

## Why deferred

The first pass optimizes for a clean, runnable, public-facing repo where users can:
- fetch/process SMEAR and SHEBA data
- run ultraspherical fits
- inspect benchmark reports and coefficient behavior

Deferred materials can be added once core reproducibility is stable.
